# Mi Portafolio

Portafolio personal desarrollado con React para mostrar mis proyectos, habilidades y experiencia en el desarrollo de software y aplicaciones móviles.

## Características

* Diseño moderno y responsive.
* Animaciones y transiciones al hacer scroll.
* Sección de presentación personal.
* Visualización de tecnologías utilizadas.
* Galería interactiva de proyectos.
* Carruseles de imágenes.
* Testimonios de clientes.
* Diseño adaptable para dispositivos móviles.

## Secciones

* Hero principal.
* Sobre mí.
* Tecnologías.
* Proyectos.
* Contacto.

## Tecnologías utilizadas

* React
* JavaScript
* HTML
* CSS
* GitHub

## Proyectos destacados

* Heliconias Report
* Prestaap
* Boletazo
